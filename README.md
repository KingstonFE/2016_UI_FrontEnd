# 2017 UI / FrontEnd
請務必依照下列虛擬情境，完成情境需求之設計
### Description
***
請設計/建置一套後台會員管理介面，此平台最少滿足以下幾點需求，此外，在需求之外也可以自行發揮想像，讓平台介面更完整
- 能做角色管理
- 能做會員管理
- 會員會被歸納在角色下面
- 每個角色能清楚了解有多少會員被歸納於此
- 每個會員能清楚被歸納到那些角色
